## Projeto Aplicado I

### Análise de Sinistros de Trânsito em Rodovias Federais Brasileiras

Este projeto tem como objetivo realizar uma análise exploratória dos dados de sinistros de trânsito registrados em rodovias federais brasileiras. A análise busca identificar padrões e fatores associados à ocorrência de acidentes, permitindo gerar insights sobre segurança viária.

Os dados utilizados são disponibilizados publicamente pelo governo brasileiro por meio do portal de dados abertos.

Fonte oficial dos dados:

PRF

Disponível em:

https://dados.gov.br/dados/organizacoes/visualizar/policia-rodoviaria-federal

Objetivos do Projeto

1. Analisar a distribuição de acidentes nas rodovias federais brasileiras
2. Identificar horários e dias com maior ocorrência de acidentes
3. Verificar as principais causas de sinistros
4. Avaliar a gravidade dos acidentes
5. Gerar insights que possam contribuir para estudos sobre segurança no trânsito
